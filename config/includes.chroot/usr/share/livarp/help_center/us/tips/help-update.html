howto update this help center

why update this help-center?

livarp_0.4 comes with a full doc inside, but the online doc may be updated for bug corrections or extra tips. update this help-center will give you lastest window managers possibilities.

livarp help center sources are available on gitorious. so that you can keep your help_center updated from git repository.

  • open a terminal and cd into your user dirctory to clone the git repository:

cd ~
git clone git://gitorious.org/livarp/livarp_04_help-center.git

  • replace as root the old help_center with the updated one:

sudo rm -R -f /usr/share/livarp/help_center && sudo mv ./livarp_04_help-center /usr/share/livarp/help_center

that's all, your livarp_0.4-help_center is up-to-date :) .
