click'me's

taking the principle of "PKGBUILDs" on Archlinux or Gentoo "ebuild", livarp integrates "clickmes", a series of scripts for easy installation package grouped together by themes.
use is simple: navigate to ~/bin/clickmes/ folder with rox-filer. once in the directory, just click on a script to run and install the relevant packages. the administrator password, you will be asked.

available click'me's

  • linux-libre: install last 100% free GNU/Linux kernel
  • dev&git pack: install git-dev package to be abble to compile sources included in livarp (dwm_reloaded, wmfs)
  • graphic suite: install gimp, inkscape, agave, xsane, imagemagick and comix
  • backup: install unison-gtk to make local or distant backups
  • lamp: install packages to activate a local web server
  • office light: install abiword and gnumeric
  • office full: install libreoffice suite and desktop wiki zim.
  • power: install laptop-detect and xfce4-power-manager for laptop power managment
  • print: install printing support
  • ssh: install openssh server/client
  • nonfree: add contrib/non-free archives and deb-multimedia repositories + install flashplugin-nonfree
