vider la corbeille

comme vous avez pu le constater, Rox-Filer n'utilise pas de corbeille, mais d'autres applications l'utilise (comme transmission…). il se peut que vous désiriez vider la corbeille de temps en temps :) il vous suffit de supprimer le dossier ~/.local/share/Trash:

avec ranger cli-manager:

  • ouvrir ranger: Alt+r
  • afficher les fichiers cachés: “zh” dans le client ranger
  • selectionner ~/.local, puis 'share' puis 'Trash'
  • effacer le répertoire: dans le client ranger, taper ”:delete” puis “y” quand il vous demande “delete seriously ?”

avec Rox-Filer:

  • ouvrir rox: Alt+Shift+r
  • afficher les fichiers cachés: Ctrl+h
  • naviguer jusqu'à ~/.local/share
  • effacer le dossier Trash en ouvrant le menu contextuel (clic gauche) puis “effacer” ou alors, sélectionner puis Ctrl+x.

avec une commande:

$ rm -f -R /home/$USER/.local/share/Trash
