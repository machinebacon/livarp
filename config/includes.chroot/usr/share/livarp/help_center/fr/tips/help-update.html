mise à jour de l'aide

pourquoi mettre l'aide à jour?

livarp_0.4 est livré avec une doc complète intégrée, mais la doc en ligne va certainement s'étoffer (corriger des bugs, ajouter des astuces). une mise à jour de cette aide vour permettra de connaitre les dernières possibilités de vos sessions.

les sources de ce centre d'aide sont disponibles sur gitorious. l'intérêt est de pouvoir mettre à jour l'aide depuis le dépôt git.

  • placez-vous dans votre répertoire utilisateur et clonez le dépôt git:

cd ~
git clone git://gitorious.org/livarp/livarp_04_help-center.git

  • remplacer en mode administrateur l'ancien centre d'aide par celui téléchargé:

sudo rm -R -f /usr/share/livarp/help_center && sudo mv ./livarp_04_help-center /usr/share/livarp/help_center

et voilà, votre centre d'aide livarp est désormais à jour :) .
