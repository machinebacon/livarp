passer GRUB

si livarp est le seul système d'exploitation présent sur votre machine, vous n'avez pas besoin des 5 secondes d'attente dans grub au démarrage de votre ordi. pour “passer grub”, il suffit de réduire le temps d'attente. pour cela, éditez le fichier /etc/default/grub:

sudo vim /etc/default/grub

puis remplacer

GRUB_TIMEOUT=5

par

GRUB_TIMEOUT=0

enregistrer et fermer avec ':wq' dans vim puis actualiser GRUB avec la commande:

sudo update-grub

ainsi, au prochain démarrage, l'écran de grub ne sera pas affiché et livarp se chargera automatiquement.

boot-up-manager

bum est un logiciel qui permet de gérer graphiquement les services lancés au démarrage de votre machine Debian sans avoir à mettre les mains dans la console.
il s'installe simplement:

# apt-get install bum

et se lance en root:

# bum

readahead

le projet vient de Fedora, le paquet sur Debian est readahead-fedora. le but est de précharger les modules nécessaires au processus de démarrage. pour l'installer, depuis un terminal sur Debian:

# apt-get install readahead-fedora

rien à faire d'autre, le démon se lance et accélère le boot.
