click'me's

reprenant le principe des "PKGBuilds" sur Archlinux ou les "ebuilds" de Gentoo, livarp intègre les "clickmes", une série de scripts facilitant l'installation de paquets réunis par thèmes.
l'utilisation est simple: naviguer jusqu'au dossier ~/bin/clickmes/ avec rox-filer. une fois dans le dossier, il suffit de cliquer sur un script pour le lancer et installer les paquets concernés. le mot de passe administrateur vous sera demandé.

liste des click'me's disponibles

  • linux-libre: installe le dernier kernel GNU/Linux 100% libre
  • dev&git pack: installe les logiciels nécessaires à la compilation des sources présentes sur livarp (dwm_reloaded, wmfs)
  • graphic suite: installe gimp, inkscape, agave, xsane, imagemagick et comix
  • backup: installe le logiciel unison-gtk pour des sauvegardes locales ou distantes
  • lamp: installe tout pour activer un serveur web local
  • office light: installe abiword et gnumeric pour un travail bureautique classic
  • office full: installe la suite libreoffice et le wiki de bureau zim.
  • power: installe laptop-detect et xfce4-power-manager pour une gestion fine de l'énergie
  • print: installe le support de l'impression
  • ssh: installe openssh server/client
  • nonfree: ajoute les dépots contrib/non-free et deb-multimedia + installe flashplugin-nonfree
