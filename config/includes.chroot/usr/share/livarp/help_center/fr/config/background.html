fond d'écran

la gestion des fonds d'écran est assurée par feh dans la configuration par défaut, et se détermine dans le script $HOME/bin/start/votre_wm_start.sh.
mais si vous désirez choisir graphiquement votre fond décran, livarp intègre le programme nitrogen.

il se lance depuis les différents menus du livarp: section configuration:fond d'écran.

si vous désirez retrouver votre fond d'écran configuré par nitrogen, vous devez éditer votre $HOME/bin/start/votre_wm_start.sh ou $HOME/.config/openbox/autostart:

  • commenter la ligne correspondant à feh (ajouter un "#")
  • décommenter la ligne concernant nitrogen (en enlevant le "#")

fond d'écran aléatoire

afin d'afficher un fond d'écran différent à chaque session, et le changer durant la session, nous allons utiliser feh. voici un petit script qui permet d'afficher un fond d'écran aléatoire issu d'un dossier spécifié en début de session, puis de changer de fond d'écran à intervalle régulier.

#! /bin/sh
# random wallpaper script
# dossier d'images
DIR="/path_to_directory"
# temps de rotation des images en secondes
INT="600"
while true; do
find $DIR -type f -name '*.jpg' -o -name '*.png' | shuf -n 1 | xargs feh --no-xinerama --bg-scale &
sleep $INT
done
exit 0

il ne vous reste plus qu'à lancer ce script dans votre fichier ~autostart.

# random wallpaper
randwalls.sh &
