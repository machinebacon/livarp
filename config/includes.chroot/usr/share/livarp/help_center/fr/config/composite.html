effets de transparence

les effets de transparence/ombrage sont gérés par xcompmgr. un script intégré, /usr/local/bin/xcompmgr_livarp, configure trois niveaux d'effets: no, léger et complet.
la configuration des effets est présente dans les différents menus du livarp; section configuration:composite.
